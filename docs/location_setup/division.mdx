# How to Create a Division

This guide explains how to create a new **Division** in the ERP system, including required fields, default values, and how divisions help manage multi-cuisine or multi-concept outlets.

---

## Step 1: Navigate to the Division Screen

1. Go to **Navigation Settings â†’ Location**.  
2. Click the **Division** icon.  
3. You will be redirected to the **Division Management** page.

![ERP Navigation Screenshot](/images/division.png)  


---

## Step 2: Create a Division

1. After creating a location, the **Division icon** will become active.  
2. Click the icon to open the **Division Tab**.  
3. Add the **default division code**:
   - If you created location code `001`, use **the same code (`001`)** for the division.
   - Leave **Price Group** empty for now (you will configure this in the next step).

### Why Divisions Are Important
Divisions are extremely useful for restaurants that operate **one physical location** but offer **multiple cuisines**, each with its own menu or product groups.  
Using Divisions, you can:

- Separate menus by cuisine  
- Manage reporting by concept  
- Control pricing and grouping efficiently  
- Run inventory per division if needed  

This makes multi-concept restaurant management smoother and more organized.
